## Findings considered and rejected

(So nobody re-audits these — see the audit for full reasoning.)

- **"Committed secrets in `.env.local`"** — false. `.env*` is gitignored and
  `git log --all -- .env.local` shows it was never committed. No action.
- **"Filter-label XSS in FilterBar"** — false. `{f.label}` renders as a React
  text node (auto-escaped); no `dangerouslySetInnerHTML` in the path. Not
  exploitable.
- **"Stale active tab in LeaderboardPanel"** — non-issue. The UI only lets you
  apply a filter from the currently-visible tab, so `active` already matches the
  filtered dimension; it can't desync via the funnel or the FilterBar.
- **"removeOne URLSearchParams fragility" / "dsTabs not keyed"** — non-issues.
  The delete-then-reappend is correct for a single multi-value key, and `Tabs`
  keys by its `value` prop.
- **Clean bills (no action):** Supabase queries are parameterized; the
  User-Agent regexes have no ReDoS; the admin auth cookie flags
  (`httpOnly`/`sameSite`/`secure`) are correct; the admin layout tree correctly
  drops analytics/consent machinery (PII-minimizing).
